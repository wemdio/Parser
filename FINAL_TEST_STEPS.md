# ФИНАЛЬНЫЙ ТЕСТ - СТРОГО ПО ШАГАМ

## ⚠️ ВАЖНО: НЕ ПРОПУСКАЙТЕ НИ ОДИН ШАГ!

### Шаг 1: Откройте браузер
Перейдите на http://localhost:3000

### Шаг 2: УДАЛИТЕ существующий аккаунт
- Найдите блок "Добавленные аккаунты"
- Нажмите кнопку **"Удалить"**
- Подтвердите удаление

### Шаг 3: ДОБАВЬТЕ аккаунт ЗАНОВО
- Нажмите кнопку **"Добавить аккаунт"**
- Введите:
  - API ID
  - API Hash  
  - Номер телефона
  - Имя (необязательно)
- Нажмите **"Добавить"**

### Шаг 4: Проверьте логи в backend
В терминале backend ДОЛЖНЫ появиться:
```
############################################################
### ADD ACCOUNT REQUEST RECEIVED ###
### Phone: +7...
############################################################

==================================================
CODE SENT SUCCESSFULLY
Phone: +79959982523
Code hash: ABC123...
Time: 02:15:30
==================================================
```

⚠️ **ЕСЛИ ЭТИХ СТРОК НЕТ - ОСТАНОВИ��ЕСЬИ СКАЖИ МНЕ!**

### Шаг 5: Дождитесь кода в Telegram
Код придет в Telegram как обычное сообщение

### Шаг 6: СРАЗУ введите код
- В течение **5-10 секунд** после получения
- Введите код в поле "Код подтверждения"
- Нажмите **"Подтвердить"**

### Шаг 7: Проверьте логи в backend
ДОЛЖНЫ появиться:
```
############################################################
### VERIFY REQUEST RECEIVED ###
############################################################

==================================================
VERIFY CODE STARTED
Phone: +79959982523
Code: 12345
Time: 02:15:45   <-- разница с предыдущим < 1 минуты!
Session file EXISTS - using saved session
==================================================

Attempting sign in...
Sign in successful!  <-- УСПЕХ!
```

## ✅ УСПЕХ если видите "Sign in successful!"

## ❌ ОШИБКА если:
- Нет строк "### ADD ACCOUNT REQUEST RECEIVED ###"
- Нет строк "CODE SENT SUCCESSFULLY"
- Разница во времени > 2 минут
- "Session file NOT FOUND"
- "Phone code expired"
