# ТЕСТ - ПОШАГОВАЯ ИНСТРУКЦИЯ

## ШАГ 1: Перезапустите backend
В терминале backend:
1. Ctrl+C (если еще не остановлен)
2. `$env:PYTHONIOENCODING='utf-8'; python run_backend.py`
3. Убедитесь что видите ">>> BACKEND STARTED <<<"

## ШАГ 2: Обновите страницу
В браузере нажмите F5

## ШАГ 3: Удалите ВСЕ аккаунты
Нажмите кнопку "Удалить" у каждого аккаунта в списке

## ШАГ 4: Добавьте НОВЫЙ аккаунт
1. Нажмите "Добавить аккаунт"
2. Введите данные
3. Нажмите "Добавить"

## ШАГ 5: Проверьте логи в backend
В терминале backend ДОЛЖНЫ появиться:
```
############################################################
### ADD ACCOUNT REQUEST RECEIVED ###
### Phone: +7...
############################################################

==================================================
CODE SENT SUCCESSFULLY
Phone: +7...
Code hash: ...
Time: 14:30:00
==================================================
```

## ШАГ 6: Введите код
1. Когда код придет в Telegram
2. Введите его СРАЗУ
3. Нажмите "Подтвердить"

## ШАГ 7: Проверьте логи в backend
ДОЛЖНЫ появиться:
```
############################################################
### VERIFY REQUEST RECEIVED ###
### Account ID: 1
### Phone code: 12345
############################################################

==================================================
VERIFY CODE STARTED
Phone: +7...
Code: 12345
Time: 14:30:15
==================================================

Sign in successful!
```

## ВАЖНО!
Если НЕ видите логи с "###" - запросы НЕ доходят до backend!
Проверьте что frontend работает на http://localhost:3000
